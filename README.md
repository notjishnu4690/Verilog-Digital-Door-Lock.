# Digital Door Lock System using Verilog HDL

## Overview

This project implements a Digital Door Lock System using Verilog HDL. The system compares an entered password with a predefined password and grants or denies access accordingly.

## Inputs

* Password Input
* Reset

## Outputs

* Door Unlock Signal
* Access Denied Signal

## Tools Used

* Verilog HDL
* Icarus Verilog
* GTKWave

## Files

* doorlock.v : Digital Door Lock design
* tb_doorlock.v : Testbench
* doorlockwaveform.jpg : Simulation waveform

## Features

* Password Verification
* Access Control Logic
* Reset Functionality

## Simulation Result

The design was successfully simulated and verified using the testbench. Waveform outputs confirm correct access-granted and access-denied responses based on the entered password.
